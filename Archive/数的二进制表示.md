#### Description
##### 整数
- 正整数采用<font color="#0593A2">除二取余法</font>
- 负整数采用<font color="#0593A2">补码</font>表示, 补码就是把正数的二进制取反再加 1

![[Pasted image 20230411112406.png]]
- 采用符号位+数据位的方式表示有符号整型
- 当数据为全为 0 时, 符号位有两种选择, 这种表示方式就会出现+0 和-0, 一般用+0 来表示 0, 而-0 来表示最小值 (-0 取反再加 1)
