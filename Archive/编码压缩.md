#### Description
在音视频领域，压缩被称为 encode（编码），因为压缩这个术语比较通用，字符串文本也能压缩，其他的二进制文件也能压缩。音视频里面的压缩算法比较有行业自身的特点，所以叫**编码**

- 常见的压缩标准: 
    1. JPEG 标准, 用于单张图片压缩
    2. H.262 标准, 用于视频编解码
    3. H.263 标准, 用于视频编解码
    4. H.264 标准, 使用最为广泛的视频编解码标准
    5. VP9, Google 的视频编解码标准
    6. AVS, 中国的视频压缩标准

- 视频编码系统中用到的主流技术: 
1. 预测
2. 变换
3. 量化
4. 熵编码
5. 环路滤波
